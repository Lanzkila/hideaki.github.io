# Manga Studio 📚

**Manga Studio** ialah aplikasi web ringkas dan pantas untuk menukar fail manga anda kepada format yang lebih mudah dibaca. Ia direka untuk memproses fail secara terus di dalam pelayar web anda tanpa perlu memuat naik fail ke pelayan pihak ketiga.

---

## 🛠️ Fungsi Utama
* **PDF Merge:** Gabungkan pelbagai fail PDF menjadi satu dokumen tunggal.
* **Convert ZIP/CBZ to PDF:** Tukar fail arkib manga (ZIP/CBZ) terus kepada format PDF.
* **Image to PDF:** Tukar koleksi imej (JPG, PNG, WebP) menjadi fail PDF.
* **Custom Control:** Pilihan untuk mengubah resolusi dan kualiti imej mengikut keperluan.

## 🚀 Cara Penggunaan
1. Layari laman utama: [https://lanzkila.github.io/hideaki.github.io/](https://lanzkila.github.io/)
2. Klik pada kawasan **Drop Zone** untuk memilih fail dari peranti anda.
3. Laraskan **Max Resolution** atau **Quality** jika perlu.
4. Tekan butang **CONVERT** untuk proses PDF, atau **REVERSE** untuk memproses CBZ.
5. Fail akan dimuat turun secara automatik ke peranti anda.

## ⚙️ Teknologi Digunakan
* [JSZip](https://stuk.github.io/jszip/) - Pengendalian fail ZIP/CBZ.
* [jsPDF](https://parall.ax/products/jspdf) - Penjanaan fail PDF.
* [pdf-lib](https://pdf-lib.js.org/) - Manipulasi PDF.
